# Font metrics calculator for font-size-adjust
This tool calculates the aspect value for an uploaded font. Use this value as your `font-size-adjust number`. For more details see [MDN](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/font-size-adjust).

Live tool available on [clagnut.com](https://clagnut.com/sandbox/font-size-adjust-ex.html).
