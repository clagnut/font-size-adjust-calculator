# Font metrics calculator for font-size-adjust
This tool calculates font metrics for an uploaded font. The values it outputs, such as aspect ratio, can be used with the CSS `font-size-adjust` property. For more details see [MDN](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/font-size-adjust).

Live tool available on [clagnut.com](https://clagnut.com/sandbox/font-size-adjust-ex.html).
